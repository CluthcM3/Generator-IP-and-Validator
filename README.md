# Generator-IP-and-Validator

```
- Generate IP Address
- Checker IP
```


### Install
Clone this project
```bash
> git clone https://github.com/CluthcM3/Generator-IP-and-Validator.git
> cd Generator-IP-and-Validator
> git clone https://github.com/WordPress/Requests.git
> composer install
```


Usage
```bash
> php generator.php
```
